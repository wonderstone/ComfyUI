# 技术库索引

这里存放的是**小而精的技术积木**，每个 workflow 专注于演示单一技术点，便于学习、调试和作为组件被引用。

## 目录结构

| 文件夹 | 说明 |
|--------|------|
| `图像处理/` | 裁剪、缩放、拼接等基础操作 |
| `提示词生成/` | 从图像提取描述（Florence、Qwen3VL） |
| `逻辑控制/` | 迭代、条件、开关等流程控制 |
| `放大修复/` | 图像放大、超分辨率 |
| `文生图基础/` | 各模型的基础文生图配置 |
| `ControlNet/` | 各类引导控制（深度、姿态、边缘等） |
| `InstantID/` | InstantID 人脸一致性技术 |
| `IPAdapter/` | IPAdapter 风格/人脸迁移技术 |
| `ReActor/` | ReActor 换脸技术 |
| `SAM3/` | Segment Anything 3 分割技术 |

## 使用说明

1. **学习用**：想了解某个技术，直接打开对应 workflow
2. **调试用**：遇到问题时，用最小化的 workflow 定位问题
3. **组件用**：复制需要的节点到你的完整 workflow 中

## 技术库 vs 目的分类

- **技术库**：回答"这个技术怎么用？"
- **目的分类**：回答"我想做 XX，用哪个 workflow？"
