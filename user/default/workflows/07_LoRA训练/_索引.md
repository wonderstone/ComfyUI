## LoRA 训练基础知识

### 什么是 LoRA？
Low-Rank Adaptation，一种轻量级微调方法，可以让模型学习特定风格、角色或概念。

### 训练数据准备
1. 收集 10-50 张目标图片
2. 图片质量要高、风格统一
3. 建议使用自动标注工具生成描述

### 常用参数
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 训练步数 | 1000-3000 | 根据数据量调整 |
| 学习率 | 1e-4 | 太高会过拟合 |
| Rank | 16-64 | 越大越精细但文件越大 |

## 常见问题

### Q: 训练后模型不像？
- 增加训练步数
- 检查训练图片质量
- 调整学习率

### Q: 训练后模型过拟合？
- 减少训练步数
- 降低学习率
- 使用正则化图片

### Q: 显存不够？
- 降低 batch size
- 使用 gradient checkpointing
- 使用 8bit 优化器

## 注意事项
- LoRA 训练需要较高显存（建议 12GB+）
- 训练时间较长，建议使用独立 GPU
- 训练完成后记得测试效果
