# 图生图 Workflow 索引

以图片为输入，进行引导生成、风格转换、放大等操作。

## Workflow 列表

| 文件 | 用途 | 显存需求 |
|------|------|---------|
| `FLUX_ControlNet_深度引导.json` | 保持构图，重新生成 | 10GB |
| `FLUX_ControlNet_深度引导V2.json` | 深度引导 V2 版本 | 10GB |
| `FLUX_ControlNet_姿态引导.json` | 保持人物姿态 | 10GB |
| `FLUX_ControlNet_深度+LoRA.json` | 深度引导 + LoRA 风格 | 12GB |

## 选择指南

### 想保持构图/布局
→ 使用**深度引导**（DepthAnything）

### 想保持人物姿态
→ 使用**姿态引导**（OpenPose）

### 想改变风格
→ 使用 ControlNet + LoRA 组合

### 想放大图片
→ 见 `_技术库/放大修复/`

## ControlNet 类型说明

| 类型 | 提取信息 | 适用场景 |
|------|---------|---------|
| Depth | 深度/远近 | 保持构图、场景重绘 |
| OpenPose | 人体骨骼 | 保持姿态、换装 |
| Canny | 边缘轮廓 | 线稿上色、保持结构 |
| DensePose | 人体表面 | 精细人物控制 |

## 常见问题

### Q: ControlNet 强度设多少？
- 0.4-0.6：轻度引导，保留创造性
- 0.7-0.9：强引导，严格遵循
- 1.0：完全遵循（可能过于死板）
